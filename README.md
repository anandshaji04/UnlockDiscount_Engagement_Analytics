# Unlock Discounts — User Acquisition & Engagement Analytics

A data analytics project built to understand how users discover and interact with **Unlock Discounts**, identify drop-off points, and generate actionable recommendations to improve traffic and engagement.

---

## 📌 Project Overview

This project analyzes **Google Analytics 4 (GA4) export data** for the Unlock Discounts website covering the period:

**January 24 – February 28, 2026**

The dataset was:
- Cleaned and restructured in **Microsoft Excel**
- Modeled and visualized using **Power BI**
- Presented across a structured **4-page interactive dashboard**

The objective was to uncover traffic trends, engagement behavior, retention patterns, and acquisition insights.

## 🔍 Key Insights

- **477 total sessions** recorded during the analysis period  
- **356 new users** and **361 active users**
- **Organic Search (45.7%)** leads session traffic
- **Direct traffic (54.2%)** drives the highest new user acquisition

### 🚨 Major Finding: Retention Problem

- Returning users are almost zero across most days  
- Week 0 cohort retention is significantly higher than Week 1–4  
- Users visit once and rarely return  

### 📉 Engagement Drop

- Average engagement starts at ~1.5 minutes on Day 0  
- Drops below 0.5 minutes within the first week  

### 📅 Engagement Pattern

- Wednesday & Thursday show peak engagement  
- Sunday has the lowest engagement levels  

### 🌍 Geographic Trends

- India leads with **229 active users**
- United States follows with **116 active users**

---

## 🎯 Business Interpretation

The platform currently performs well in **user acquisition**, particularly through Organic and Direct channels. However, it struggles significantly with:

- User retention  
- Repeat engagement  
- Sustained session duration  

Traffic is being generated, but users are not converting into repeat visitors.

---

## 🚀 Recommendations

1. **Improve Retention Strategy**
   - Introduce remarketing campaigns
   - Use email re-engagement flows
   - Push personalized offers to first-time users

2. **Optimize Landing Experience**
   - Improve first-session engagement
   - Reduce bounce points
   - Add clearer discount value propositions

3. **Leverage Peak Engagement Days**
   - Run promotional campaigns on Wednesdays & Thursdays

4. **Geographic Targeting**
   - Focus paid campaigns in India and the US
   - Localize offers for high-performing regions

5. **Cohort-Based Marketing**
   - Design retention strategies specifically targeting Week 1 drop-offs
